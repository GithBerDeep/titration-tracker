# Titration Tracker (PWA)

Local-first ADHD medication titration tracker.

Designed to:
- Record real-world duration of effect
- Capture perceived benefit and crash intensity
- Generate clinician-friendly reports
- Keep all data stored locally (no server)

## Status

Experimental. Personal use during medication titration phase.

## Features (v0.1)

- Timestamp-based intake and effect end
- Manual correction mode
- Benefit (0–10) and crash (0–10) tracking
- Basic side effects checkboxes
- JSON / CSV export
- Printable clinician report
- Offline support (PWA)

## Data & Privacy

All data is stored locally in the browser.
No server, no tracking, no analytics.
